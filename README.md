# Şəxsi Portfolio Veb-saytı

Frontend Tərtibatçısı olaraq hazırladığım şəxsi portfolio veb-saytım. Bu layihə mənim texniki bacarıqlarımı, fərdi layihələrimi və proqramlaşdırmaya olan yanaşmamı potensial işəgötürənlərə və komanda rəhbərlərinə nümayiş etdirmək üçün nəzərdə tutulub.

## 🛠️ Texnologiyalar və Kitabxanalar

*   **Nüvə (Core):** React.js
*   **Dil:** JavaScript (ES6+)
*   **Vizual Dizayn:** Tailwind CSS
*   **UI Komponentləri:** Material UI (MUI) (Daha əlavə olunacaq)
*   **Animasiyalar:** Framer Motion
*   **Paket Meneceri:** Npm
*   **İnşa Aləti (Build Tool):** Vite

## ✨ Özəlliklər

*   **📱 Tam Responsiv Dizayn:** Bütün cihazlarda vizual olaraq optimizasiya olunub (Bəzi Planşetlərdə Problem Yarana Bilər).
*   **🎨 Müasir UI/UX:** Tailwind CSS-in çevikliyinin mükəmməl kombinasiyası.
*   **✨ Hamar Keçidlər:** Framer Motion vasitəsilə veb-sayta gözoxşayan yumuşaq keçid effekti verilib.
*   **📂 Dinamik Layihələr Bölməsi:** Hazırladığım digər şəxsi layihələrin səliqəli nümayişi.

## 📦 Layihənin Yerli Kompüterdə Başladılması

Layihəni öz lokal mühitinizdə yoxlamaq üçün terminalda ardıcıl olaraq aşağıdakı əmrləri icra edin:

1. Layihəni klonlayın:
   git clone https://github.com/Appleex0/My-Portfolie-Site.git

2. Layihə qovluğuna keçid edin:
   cd My-Portfolie-Site

3. Lazımi paketləri (Material UI və Framer Motion daxil olmaqla) yükləyin:
   npm install

4. Layihəni lokal serverdə başladın:
   npm run dev
